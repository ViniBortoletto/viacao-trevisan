# Viação Trevisan

## Ferramentas usadas

- HTML5
- CSS3
- SASS
- JavaScript
- Slick Carousel
- Gulp

Live Preview: https://viacao-trevisan.netlify.com/
Site original: https://viacaotrevisan.com.br/

## Instalação e uso

Clone ou baixe o repositório. Abre o terminal, navegue até a pasta e excute o comando:

```bash
npm install && gulp
```
